Packages available for ngspice

Packages are ready-to-install archives. Ngspice packages are made available by the ngspice team and several partners.

Official ngspice packages

The ngspice team offers packages for MS Windows and a binary for macOS. For other OSs you may download the sources and compile them yourself, from the ngspice file release page.

Packages in distributions

Several software distributions make compiled and packaged ngspice available, ready for installation with a suitable package manager. Packages may offer the ngspice exe, the shared ngspice library or the tcl ngspice library version. Please check below if your distribution has a ngspice package ready for you.

Gentoo Linux
Gentoo Linux offer ngspice-27 in the sci-electronics category.
  • Packages page
Debian
Ngspice has been packaged for Debian Linux "sid", "bullseye", and "buster backports", version 31.3-1.
  • Package description page
Ubuntu Linux
Ubuntu Linux provides an ngspice package (version 31.3-1) in their "eoan" distributions in the "electronics" section.
  • Package description page
Arch Linux
Arch Linux provides an up-to-date ngspice package (version 31-1) in their actual distribution.
  • 64 Bit package description page
Fedora
Fedora provides the ngspice package (version 31.3) in their F30, F31, and F32 distributions.
  • ngspice build page
openSUSE
openSUSE provides an ngspice package (version 30) in all of their distributions.
  • ngspice description page
Cygwin
Cygwin provides an updated ngspice package (ngspice-31).
  • Cygwin home page
FreeBSD
FreeBSD provides a ngspice port in their "cad" section (version 30).
  • Port description page

The list above is surely incomplete. If you know of other ngspice packages, please give details to the ngspice-users forum .
