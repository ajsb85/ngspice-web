Packages available for ngspice

Packages are ready-to-install archives. Ngspice packages are made available by several partners.

Official ngspice packages

The ngspice team does not maintain or offer specific packages, except a binary for MS Windows. Instead you may download the sources, which compile on many OSs, from the ngspice file release page.

Packages in distributions

Some distributions may have packaged ngspice for you, if you do not want to get the source and compile them, check below if your distribution has a ngspice package ready for you.

Gentoo Linux
Gentoo Linux offer ngspice-27 in the sci-electronics category.
  • Packages page
Debian
Ngspice has been packaged for Debian Linux "sid", version 27-1.
  • Package description page
Ubuntu Linux
Ubuntu Linux provides an ngspice package (version 27-1) in their "bionic" distributions in the "electronics" section (in the "multiverse", due to ngspice license).
  • Package description page
Arch Linux
Arch Linux provides an up-to-date ngspice package (version 28-1) in their actual distribution.
  • 64 Bit package description page
openSUSE
openSUSE provides an ngspice package (version 27) in their Tumbleweed distribution.
  • ngspice description page
Cygwin
Cygwin provides an up-to-date ngspice package (ngspice-28).
  • Cygwin home page
FreeBSD
FreeBSD provides a ngspice port in their "cad" section. This is an old release however, with ngspice version 26.
  • Port description page

The list above is surely incomplete. If you know of other ngspice packages, please give details to the ngspice-users forum .
