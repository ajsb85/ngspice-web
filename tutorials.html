Ngspice Tutorials

The first entry is a tutorial dedicated to ngspice newbies. Starting from download and installation, the basic simulation types are presented. The ngspice manual does not contain introductory chapters, but finally will give you details on all the features offered by ngspice. If you want a GUI with schematic capture, we recommend KiCAD. An introductory tutorial for ngspice in KiCAD is found in the third list entry.

  • Ngspice tutorial for beginners.

  • Ngspice user's manual (updated).

  • Ngspice using KiCad/Eeschema GUI for schematic entry, simulation and plotting.
